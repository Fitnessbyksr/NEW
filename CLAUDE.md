# Email Marketing Funnel System

A complete AI-powered email marketing funnel toolkit with 10 specialized skills that work together to build, grow, and monetize an email list.

## Skills (Slash Commands)

| Command | Purpose |
|---|---|
| `/positioning-angle` | Find your unique value proposition and market angle |
| `/lead-magnet` | Design high-converting lead magnets to grow your list |
| `/direct-response-copy` | Write persuasive hooks, headlines, and sales copy |
| `/email-sequences` | Build automated drip campaigns and nurture sequences |
| `/keyword-research` | Discover SEO/GEO topics that drive organic traffic |
| `/seo-content` | Write long-form articles that rank and convert |
| `/newsletter-skill` | Create engaging newsletters for ongoing communication |
| `/content-atomizer` | Repurpose content into video scripts, social posts, and more |
| `/brand-voice` | Codify your writing style and voice guidelines |
| `/orchestrator` | Get guided help choosing the right skill for your goal |

## Recommended Workflow

1. Start with `/orchestrator` if you're unsure where to begin
2. Run `/positioning-angle` first to lock in your market position
3. Build your `brand-voice` so all output stays on-brand
4. Create a `/lead-magnet` to start collecting emails
5. Write `/direct-response-copy` for your opt-in pages and ads
6. Set up `/email-sequences` for automated follow-up
7. Use `/keyword-research` + `/seo-content` to drive organic traffic
8. Send `/newsletter-skill` campaigns to engage your list
9. Feed everything into `/content-atomizer` to multiply your output

## Project Context

Store your brand/niche details in `context/brand.md` so every skill can reference them automatically.
